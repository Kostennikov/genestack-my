Omics Data Manager

Catalogue, curate, search multi-omics data at scale.

Big data needs better digital health. ODM makes it easy to integrate, harmonise and search omics data from multiple sources making it findable, accessible, interoperable and reusable.

A collaborative data ecosystem

We have specific APIs for different user classes, such as data managers, bioinformaticians and discovery
scientists, so that everybody can harness the power of multi-omics data in their day to day work.

Amy Tang, PhD, ODM Product Manager
